# AURUM SIGNALS

AURUM SIGNALS is a personal gold (XAU/USD) analysis & signal platform. This is Level 1 (INC-0): a running monorepo skeleton (NestJS 10 backend + Next.js 14 frontend) and the complete Supabase (Postgres 15) database schema — no indicators, ingestion, signal logic, or alerts yet.

**Analysis tool, not financial advice. Past performance does not guarantee future results. Personal use only.**
